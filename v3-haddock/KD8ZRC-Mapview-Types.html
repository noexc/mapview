KD8ZRC.Mapview.Types

Description

Provides fundamental types intrinsic to the rest of the mapview library. This shouldn't contain types specific to any particular use of the library (suggested types can be found in the KD8ZRC.Mapview.Utility.* set of modules).

data MapviewConfig t

The configuration for this instance of Mapview. The t parameter is the type that the telemetry parser parses into, if it is successfully able to parse the data.
